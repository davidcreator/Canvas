Breakout Clássico

Pontos: 0
Vidas: 3
Nível: 1

Como Jogar:

• Use as setas ← → ou A/D para mover a raquete

• Quebre todos os blocos para avançar de nível

• Não deixe a bola cair!

Conceitos de Programação Aplicados:

  • Canvas API: Renderização 2D em tempo real
  • Game Loop: Ciclo de atualização e renderização
  • Detecção de Colisão: Física básica de jogos
  • Estados do Jogo: Gerenciamento de diferentes telas
  • Eventos de Teclado: Input do jogador
  • Orientação a Objetos: Classes para entidades do jogo
