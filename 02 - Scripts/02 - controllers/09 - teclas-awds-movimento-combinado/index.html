🎮 Jogo do Quadrado Controlável

Instruções: Certifique-se de que esta janela tenha foco, então use as teclas A, D, W e S no teclado para mover e rotacionar o quadrado vermelho.

Dica: Quando pressionar simultaneamente as teclas A e S, o quadrado vermelho se moverá para baixo e rotacionará para a esquerda.

💡 Para Professores: Este código demonstra conceitos de Canvas HTML5, Game Loop, Eventos de Teclado, Trigonometria e Orientação a Objetos.

📝 Códigos das Teclas: A=65, D=68, W=87, S=83
