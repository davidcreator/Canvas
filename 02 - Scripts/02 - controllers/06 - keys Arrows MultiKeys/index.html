🔍 Como o código funciona:

1. Canvas: É como uma tela em branco onde desenhamos

2. Componente: O quadrado vermelho é um "objeto" com propriedades (tamanho, cor, posição)

3. Loop do jogo: A cada 16ms (60 FPS), o jogo limpa a tela e desenha tudo novamente

4. Eventos de teclado: Capturamos quando as teclas são pressionadas/soltas

5. Física básica: Posição + Velocidade = Nova Posição
