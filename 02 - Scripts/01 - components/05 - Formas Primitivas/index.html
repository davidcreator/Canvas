🎨 Formas Primitivas Canvas

O que este código faz:

Este exemplo demonstra como desenhar diferentes formas primitivas usando Canvas HTML5: retângulos, círculos, elipses, triângulos, linhas, polígonos e formas customizadas!

💡 Formas disponíveis:

🟩 Retângulo
Forma básica com 4 lados
🟨 Quadrado
Retângulo com lados iguais
🔴 Círculo
Forma redonda perfeita
🟣 Elipse
Círculo achatado
🔺 Triângulo
Forma com 3 lados
⭐ Estrela
Polígono com pontas
📐 Linha
Traço reto
🔸 Losango
Quadrado rotacionado
