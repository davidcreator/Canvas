🎮 Jogo do Personagem Animado






📋 Como funciona este jogo:

• Movimento: Use os botões para mover o personagem pela tela

• Cenário: O fundo se move automaticamente criando sensação de movimento

• Mudança de visual: O personagem muda de expressão quando se move

• Cenários: Você pode trocar o fundo usando o menu suspenso

• Loop infinito: O fundo se repete continuamente para criar movimento perpétuo

• Fallback visual: Se as imagens não carregarem, o jogo usa formas coloridas

📁 Estrutura de Arquivos Necessária:

• Pasta "images/" deve conter:

  - smiley.gif (personagem normal)

  - angry.gif (personagem em movimento - opcional)

  - citymarket.jpg (cenário mercado)

  - forest.jpg (cenário floresta)

  - desert.jpg (cenário deserto)
