⚙️ Configurações do Jogo

Como Jogar Damas:

  • Objetivo: Capture todas as peças do adversário ou bloqueie seus movimentos.
  • Movimento: Peças se movem diagonalmente para casas vazias.
  • Captura: Pule sobre peças adversárias para capturá-las.
  • Dama: Ao chegar na última fileira, a peça vira dama e pode se mover para trás.
  • Captura Obrigatória: Se você pode capturar, deve fazê-lo!

Modalidades de Jogo:

  • Clássico: Dois jogadores locais seguindo regras tradicionais
  • Contra IA: Desafie diferentes níveis de inteligência artificial
  • Cronometrado: Cada jogador tem tempo limitado para fazer seus movimentos
  • Tutorial: Aprenda com dicas e sugestões durante o jogo

Conceitos de Programação (Para Estudantes):

  • Canvas API: Desenho de gráficos 2D em JavaScript
  • Estruturas de Dados: Array 2D para representar o tabuleiro
  • Detecção de Eventos: Mouse clicks e coordenadas
  • Lógica de Jogo: Estados, validação de movimentos
  • IA Básica: Algoritmos de decisão para jogadores artificiais
  • Sistema de Turnos: Controle de tempo e alternância
